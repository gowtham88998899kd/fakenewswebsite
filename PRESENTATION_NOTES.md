# Presentation Notes

## What to Say First

My project title is **Fake News Classifier Using Machine Learning**.  
This project checks whether a given news paragraph looks **real** or **fake**.  
I developed this project using **Python, Flask, HTML, CSS, TF-IDF, and Logistic Regression**.

## Project Flow

1. User pastes the news content.
2. The system cleans and processes the text.
3. TF-IDF converts text into numerical features.
4. Logistic Regression predicts the label.
5. The website shows the result as **REAL** or **FAKE**.

## Why I Chose This Project

- Fake news is a major problem in social media.
- This topic is useful and relevant.
- It is a practical application of AI and ML.
- It is easy to demonstrate in front of faculty.

## Key Points to Explain

- **Frontend**: HTML and CSS are used to create the website.
- **Backend**: Flask handles user input and prediction.
- **Machine Learning**: The model is trained on sample real and fake news text.
- **Output**: The website gives the prediction and confidence percentage.

## If They Ask About Algorithm

I used:

- **TF-IDF Vectorizer** to convert text into numbers.
- **Logistic Regression** to classify the text into real or fake.
